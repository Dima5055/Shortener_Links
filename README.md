# Cокращатель ссылок

## ТЗ
- Создание базы данных на sqlite с использованием конфигурации и консоли фреймворка Symfony.
- Использование Docktrine ORM. Работа с сущностями, репозиториями, миграциями.
- Реализация всех необходимых CRUD операций для работы с ссылками: 
  - добавление новой ссылки, 
  - удаление выбранных ссылок, 
  - получение полной ссылки по сгенерированному краткому идентификатору, 
  - изменение даты последнего обращения и счётчика использований,
